# Privacy Policy

Last updated: March 9, 2026

## Summary

All Tabs Closure does **not** collect, sell, share, or transmit your personal data.

## Data Collection

This extension collects **no personal information** and **no browsing content** for analytics or advertising.

The extension does not:

- create user accounts
- send usage telemetry
- track you across websites
- send your tab list, history, or identifiers to external servers

## Data Storage

The extension stores only local settings needed to function:

- selected close mode
- selected UI theme

These settings are stored in browser extension storage on your device.

## Permissions

- `tabs`: used only to perform tab actions requested by you.
- `storage`: used only to save your selected settings.

## Network Access

The extension does not require remote APIs to operate and does not upload your data.

## Changes

If this policy changes in the future, the updated version will be published in this repository.
